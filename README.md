# Binary Search Tree

Binary search tree implemented in JavaScript.

## Installation

`npm install --save-dev bst-extended`

## Usage

Import `bst-extended` module to your code:

```javascript
const BST = require('bst-extended');
```

or with ES6 syntax:

```javascript
import BST from 'bst-extended';
```

## Tests

`npm test`
